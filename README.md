# Prime
在HP Prime上运行的好玩的东西.

程序在实体机G1(V2)上调试完成，未标注**后门**者，应当在G1和G2上都能运行. 官方模拟器则未知.

## 版本问题
`G1`=`V1（硬件版本为A）和V2（硬件版本为C）`, G2则是单独的（硬件版本为D）

## 注意事项
运行程序前请先查看当前的运行内存. 运行内存过低可能会导致计算器重启并丢失数据.

不要尝试让计算器出现`初始化变量时出错`，例如用AFiles访问未定义的变量。否则，你的计算器存储空间可能会变为`0MB`.
